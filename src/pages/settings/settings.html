1. Upstairs section allows for results between 0 and 5 for each row, multiplied by the row value: 1-1-1 would result in 3 for 1st row.
2. Upstairs final result adds 50 points, if the sum of all rows equals or exceeds 63 points.
3. Straights can be marked by X, as their value is always fixed.
4. Yatzy gives a fixed amount of 50 points.
5. Rows that are being skipped are marked with '-' (or 0). Special features for Alternative Nordic yatzy
1. Upstairs section allows for results between -5 to +5 for each row, multiplied by the row value: 1-1-1 would result in 0 or X for 1st row. 2-2 would result in -2. 6-6-6-6 would result in 6 (or +6).
2. Upstairs final result gives 50 points, if the sum of all rows equals or exceeds 0 points. Any extra points are added to the result. For example 0 for ones, +2 for for twos, -6 for threes, 0 for fours, +5 for fives and +6 for sixes, result in +7 as final result, which would become 57 in the upstairs total. If sum is below 0, the upstairs result is 0.
3. Straights can be marked by X, as their value is always fixed.
4. Yatzy gives a 55 points for yatzy of ones, 60 for yatzy of twos. Add 5 points for each number. Yatzy of sixes is 80 points.
5. Rows that are being skipped are marked with '-' (or 0).
